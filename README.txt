Это проект программы архиватора.
На вход программы поступает путь до каталога, который должен быть заархивирован, а на выходе файл, содержащий данные всех файлов в этом каталоге и подкаталогах.
Также у программы должна быть возможность разархивировать этот файл.

Для запуска в режиме архиватора используется команда: ./myzip -a "путь к папке которую архивируем" -o "путь к папке куда кладем файл архива, и название архива(например /home/alex/Study/arch)"
Для запуска в режиме разархиватора используется команда: ./myzip -u "путь к архиву" -o "путь куда разархивируем"

Формат файла архива предполагается сделать как в следующем примере:
name_fold1
	<
	name_file1|size_file1 data_file1
	name_file2|size_file2 data_file2
	name_fold2
		<
		name_file3|size_file3 data_file3
		name_file4|size_file4 data_file4
		>
	>
	
(здесь для лучшей читаемости используются табуляции, пробелы и переходы на новую строку, в реальном файле архива их быть не должно)
